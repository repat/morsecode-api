Documentation for morsecode-api.de

This website documents the installation and usage of morsecode-api.de, a side-project I created to create and play around with REST-APIs, node.js and Heroku.

GET /encode

Request

Submit one parameter string e.g.

$ curl http://morsecode-api.de/encode/LOVE

Response

The Response has 2 attributes: plaintext and morsecode, e.g.

{
  "plaintext":"LOVE",
  "morsecode":".-.. --- ...- ."
}

GET /decode

Request

Submit one parameter string and URL-encode the white spaces to %20 e.g.

$ curl http://morsecode-api.de/decode/....%20.-%20-%20.

Response

The Response has 2 attributes: plaintext and morsecode, e.g.

{
  "morsecode":".... .- - .",
  "plaintext":"HATE"
}

Sourcecode and deployment

Get the sourcecode from GitHub.

$ mkdir morsecode-api
$ cd morsecode-api
$ git clone https://github.com/repat/morsecode-api.git

To start the app, change the port variable and run

$ nodemon server.js

As you might have noticed, the Procfile file is an indicator that this code could also be run on Heroku. Assuming that the Heroku Toolbelt is installed and the repository is prepared you can follow the normal instruction for deploying an app. Make sure that the port variable is set to process.env.PORT.
